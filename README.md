# CeramicsStudio
A Co-Op Ceramics Studio page that includes artists' pages, classes and more!

CeramicsStudio is a community-oriented website designed for a collaborative ceramics workshop. It includes:

 -Artist pages with personal portfolios

-Classes and skill-building events calendar

-Studio details with location and contact info

-Responsive and modern UI (e.g. add/remove features)

Tech Stack: JavaScript, CSS, HTML, MongoDB and React framework.
